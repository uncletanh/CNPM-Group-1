# NovaChat AI - Frontend Admin Dashboard

## Hướng dẫn chạy dự án

### Bước 1: Cài đặt thư viện Node.js
Mở terminal tại thư mục `frontend` và gõ:
```bash
npm install
```

### Bước 2: Chạy giao diện (Development Mode)
```bash
npm run dev
```
Trang web sẽ tự động bật tại `http://localhost:5173`.
*Chú ý: Hãy đảm bảo đã chạy cả Backend ở cổng 8000 để việc gọi API không bị lỗi.*

### Tài nguyên tham khảo:
- Sử dụng **Tailwind CSS** để style: https://tailwindcss.com/
- Truy cập vào file `src/services/api.ts` để xem cấu hình Axios.
